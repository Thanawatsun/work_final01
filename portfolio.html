ความรู้ต่างๆที่ได้มาใน 1 เทอม

 -ใน 1 เทอมที่แล้ว ผมเขียน web ไม่เป็นเลย แต่หลังจากได้เรียนวิชานี้
    และการต้องใช้ในวิชาต่างๆ มันทำให้ผมได้เรียนรู้และการศึกษาด้วยตัวเอง
    ทำให้ผมมีความมั่นใจในความสามารถตัวเองในการเขียน web

 - หลังจากได้เรียนเขียน web แล้ว ผมมีความรู้สึกว่า งานเขียน web 
    เป็นงานที่กินเวลามาก ต้องใช้เวลา และการลองออกแบบ 
    เพื่อให้งานออกมาดีที่สุด

 - ใช้ z-index ในการซ่อนข้อความ และจะใช้ onclick ในการแสดงออกมา
