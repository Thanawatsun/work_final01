ความรู้ต่างๆที่ได้มาใน 1 เทอม

-ใน 1 เทอมที่แล้ว ผมเขียน web ไม่เป็นเลย แต่หลังจากได้เรียนวิชานี้ และการต้องใช้ในวิชาต่างๆ มันทำให้ผมได้เรียนรู้และการศึกษาด้วยตัวเอง ทำให้ผมมีความมั่นใจในความสามารถตัวเองในการเขียน web

- หลังจากได้เรียนเขียน web แล้ว ผมมีความรู้สึกว่า งานเขียน web เป็นงานที่กินเวลามาก ต้องใช้เวลา และการลองออกแบบ เพื่อให้งานออกมาดีที่สุด

เทคนิคการเขียนเว็บ

- เทคนิคส่วนใหญ่ที่ใช้ในการเขียนทุกๆเว็บคือ position เพราะง่าย และจัดโครงสร้างได้ดี

- ใช้ z-index ในการซ่อนข้อความ และจะใช้ onclick ในการแสดงออกมา

- ใช้อินเตอร์เน็ตในการหารูปแบบการทำงานของ JavaScript ที่เราต้องการ แล้วนำมาปรับใช้กับงาน

- ประการค่าที่เราใช้บ่อยๆเป็น var เพื่อความง่ายในการใช้งานและจดจำ

- หากสามารถกำหนดความกว้าง,ยาว,สูงเป็น % ได้แนะนำให้กำหนดเป็น % เพราะจะแสดงผลบนจอเล็ก สมส่วนเหมือนจอใหญ่

การให้กำลังใจตัวเอง

"หากงานเสร็จไวเราก็จะได้พักไว" นี้คือสิ่งที่ผมบอกกับตัวเองมาเสมอ เพื่อให้กำลังใจตัวเองในการทำงาน โดยมีรางวัลเป็นเวลาว่าง เอาไว้ทำในสิ่งที่อยากทำในสิ่งที่ตัวเองชอบ

ได้รับอะไรบ้างจากวิชานี้

สิ่งที่ผมได้รับจากวิชานี้คือ การออกแบบเว็บ, การวาดภาพจาก vector, การใช้คำสั่งของ JavaScript และการหาข้อมูลทางอินเตอร์เน็ต เพราะ JavaScript ทำงานได้เยอะมากเลยต้องหาข้อมูลจากอินเตอร์เน็ต

สิ่งที่ผมชอบที่สุดของวิชานี้คือ SVG เพราะการวาดภาพด้วยอุปกรณ์ต่างๆ ไม่ว่าจะเป็น path หรือ polygon มันสามารถสร้างสรรค์ภาพออกมาได้ตามใจนึก ด้วยชุดตัวเลขเท่านั้น
